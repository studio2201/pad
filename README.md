# Pad

Collaborative real-time scratchpad built in Rust.

## Quick Start

### Self-Hosting (Docker)
Pull and run the official Docker container:
```bash
docker run -d -p 4402:4402 -v /path/to/appdata:/app/data ghcr.io/etecoons/pad:latest
```
