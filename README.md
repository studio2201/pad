# <img src="assets/icon.png" width="32" height="32" valign="middle"> Pad

[![CI](https://github.com/studio2201/pad/actions/workflows/ci.yml/badge.svg)](https://github.com/studio2201/pad/actions/workflows/ci.yml)

Collaborative real-time scratchpad built in Rust.

## Quick Start

### Self-Hosting (Docker)
Pull and run the official Docker container:
```bash
docker run -d -p 4402:4402 -v /path/to/appdata:/app/data ghcr.io/studio2201/pad:latest
```
